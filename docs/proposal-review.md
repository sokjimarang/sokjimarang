# "속지마랑" 기획안 비판적 검토

> 작성일: 2025.01.09
> 대상 문서: 속지마랑 - 디벨롭 기획안 v0.3 (PoC)

---

## 개요

본 문서는 Voice AI 기반 보이스피싱 예방 교육 서비스 "속지마랑" 기획안에 대한 비판적 검토 결과를 정리한 것입니다.

---

## 1. 법적/윤리적 리스크 (심각도: 높음)

### 우려사항

| 항목 | 문제점 |
|------|--------|
| **Voice Cloning 법적 이슈** | ElevenLabs Voice Cloning을 사용해 "사기범 목소리"를 만드는 것은 역설적으로 보이스피싱 범죄에 악용될 수 있는 기술. 서비스 자체가 범죄 도구로 전용될 위험 |
| **통화 녹음 동의** | 교육 목적이라도 통화 녹음 시 개인정보보호법상 명시적 동의 필요. 고령층 대상 동의 획득 절차가 명확하지 않음 |
| **심리적 충격** | 고령층에게 "속는 경험"을 제공하는 것이 트라우마나 불안을 유발할 가능성. 심리 안전장치 미비 |
| **사칭 관련 법률** | AI가 공공기관(검찰, 경찰, 금감원) 사칭 시나리오를 수행할 경우, 교육 목적이라도 법적 검토 필요 |

### 보완 제안

- **법률 검토**: 변호사/법무팀을 통한 서비스 적법성 사전 검토 필수
- **윤리 가이드라인**: Voice Cloning 기술 사용 범위 및 제한 명문화
- **심리 안전장치**: 통화 시작 전 "훈련임을 인지"시키는 옵션, 통화 중 즉시 종료 키워드(예: "훈련 중단") 설계
- **동의 프로세스**: 사전 동의서 설계 (녹음, 교육 목적 사용 등)

---

## 2. 핵심 가설 검증 부재 (심각도: 높음)

### 우려사항

| 항목 | 문제점 |
|------|--------|
| **교육 효과 미검증** | "체험 기반 학습이 효과적이다"는 핵심 가설에 대한 근거 없음. 기존 연구나 파일럿 데이터 부재 |
| **타겟 니즈 미확인** | 50대 이상 고령층이 실제로 이런 서비스를 원하는지, 수용 가능한지 검증 없음 |
| **결제자 ≠ 사용자** | 자녀가 결제하고 부모가 사용하는 구조에서 발생하는 friction 고려 안됨 |

### 보완 제안

- **PoC 범위 재정의**: 기술 검증만이 아닌 "사용자 수용성 검증"을 PoC 목표에 포함
- 5~10명 정도의 소규모 사용자 테스트를 PoC에 포함
- 기존 보이스피싱 예방 교육 효과 연구 자료 조사 및 인용

---

## 3. 기술적 리스크 (심각도: 중간)

### 우려사항

| 항목 | 문제점 |
|------|--------|
| **한국어 품질** | Whisper, Gemini, ElevenLabs 모두 영어 대비 한국어 품질이 낮음. 특히 고령층 발화(사투리, 불명확한 발음) 인식률 미지수 |
| **지연시간 낙관적** | "목표: <1초"이지만, 네트워크 지연 + 각 컴포넌트 지연 합산 시 실제로 달성 가능한지 불확실 |
| **Vapi 종속성** | Vapi에 전적으로 의존하는 구조. Vapi 장애 시 서비스 전체 마비, 가격 정책 변경 시 대응 어려움 |
| **전화번호 문제** | "한국 전화번호 제외"라고 했지만, MVP에서 070/010 번호 획득 방안이 구체적이지 않음 |

### 보완 제안

- **고령층 음성 샘플 테스트**: PoC에서 실제 고령층 음성으로 STT 정확도 측정
- **지연시간 벤치마크**: 목표치와 함께 "허용 가능 최대치" 정의 (예: <1.5초)
- **Vapi 대안 조사**: Retell.ai, Bland.ai 등 대안 플랫폼 비교 분석
- **한국 전화번호**: 통신사 API(KT, SKT), 가상 번호 서비스(070) 사업자 사전 접촉

## 4. 비즈니스 모델 부재 (심각도: 중간)

### 우려사항

| 항목 | 문제점 |
|------|--------|
| **수익 모델 미정의** | 누가, 얼마를 내는지 전혀 언급 없음 |
| **단가 계산** | 통화당 $0.10/분 × 5분 = $0.50(≈700원). 1회 교육에 얼마를 받을 수 있는가? |
| **반복 사용성** | 교육 특성상 1~2회 체험 후 재사용 동기 낮음. LTV(고객생애가치) 확보 전략 부재 |
| **경쟁 환경** | 금융감독원, 은행, 통신사의 무료 보이스피싱 예방 서비스와의 차별점 불명확 |

### 보완 제안

- **B2B 모델 구체화**: 금융기관, 지자체, 통신사 대상 기업용 서비스 모델 설계
- **B2C 가격 테스트**: 적정 가격대 조사 (무료 vs 유료, 구독 vs 건당)
- **경쟁 분석 추가**: 기존 보이스피싱 예방 교육 서비스 벤치마킹

---

## 5. 시나리오/콘텐츠 전략 미흡 (심각도: 중간)

### 우려사항

| 항목 | 문제점 |
|------|--------|
| **시나리오 깊이 부족** | "단일 시나리오"만 언급. 어떤 유형의 보이스피싱인지, 대화 플로우가 어떻게 설계되는지 불명확 |
| **사용자 반응 분류** | "수용/의심/거부" 3가지만으로 복잡한 대화 상황을 커버할 수 있는지 의문 |
| **디브리핑 설계** | 교육 효과의 핵심인 디브리핑이 "결과 요약, 탐지 포인트, 대응 수칙" 수준으로만 언급 |

### 보완 제안

- **시나리오 설계서 별도 작성**: 최소 1개 시나리오의 상세 대화 플로우 다이어그램
- **전문가 자문**: 경찰청, 금감원의 보이스피싱 사례 분석 자료 활용
- **디브리핑 UX 설계**: 통화 직후 vs SMS/웹 링크 등 전달 방식 구체화

## 6. 로드맵 구체성 부족 (심각도: 낮음)

### 우려사항

| 항목 | 문제점 |
|------|--------|
| **일정 미정** | 각 Phase별 예상 소요 기간 없음 |
| **자원 계획 없음** | 필요 인력, 예산 규모 미언급 |
| **마일스톤 불명확** | PoC → MVP 전환 기준이 "작동 확인"뿐 |

### 보완 제안

- **PoC 일정**: 2~4주 단위로 세부 일정 수립
- **Go/No-Go 기준**: PoC 결과에 따른 의사결정 기준 명문화

## 핵심 인사이트

### PoC의 함정
기술적 작동 여부만 검증하는 PoC는 "만들 수 있다"만 확인하고 "만들어야 하는가"를 놓칠 수 있습니다. 사용자/시장 검증을 병행해야 합니다.

### 양날의 검 기술
Voice Cloning + 보이스피싱 시뮬레이션은 범죄 악용 가능성이 있어, 서비스 출시 시 사회적 논란이 될 수 있습니다. 선제적 커뮤니케이션 전략이 필요합니다.
